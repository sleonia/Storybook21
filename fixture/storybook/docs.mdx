# Docs

#### Introduction
Storybook21 is a tool for UI development. It makes development faster and easier by isolating components. This allows you to work on one component at a time. You can develop entire UIs without needing to start up a complex dev stack, force certain data into your database, or navigate around your application.

#### Which commands we provide ðŸ¤–?
- `start` - for run app with <a href="https://webpack.js.org/configuration/dev-server" target="_blank">webpack-dev-server</a> on default port (4242) or your custom port
- `build` - build app to static files for nginx server, for example
- `init` - create default config file and content folders for quick start without boilerplate code
<br />
For show all options and commands use:
```bash
npx storybook21 --help
```

#### How show prop types?
We provide `PropTypes` component.
For show props you need to specify path to component in `component` field in object in navigation.
Use props _displayName_ for show props-table. If _displayName_ is incorrect you'll get error in console.

```tsx
<PropTypes displayName="YOUR_COOL_COMPONENT_NAME" />
```

Under the hood we use <a href="https://github.com/styleguidist/react-docgen-typescript" target="_blank">react-docgen-typescript</a> for typescript files and <a href="https://github.com/reactjs/react-docgen" target="_blank">react-docgen</a> for javascript files.

Show examples [ts props](/components/ts-props-example) or [js props](/components/js-props-example)

#### How to use context in your components?
storybook21 provide DocsContext wich contains:
```tsx
type DocsContextType = {
    toggleColorScheme: () => void // toggle theme light -> dark and dark -> light
    colorScheme: 'light' | 'dark'
}
```
you can use it in your components like this:
```tsx
import { DocsContext } from 'storybook21/src/theme-provider/docs-provider'

export const ContextExample = () => (
    <DocsContext.Consumer>
        {({ toggleColorScheme, colorScheme }) => (
            <Button
                variant="outline"
                onClick={() => toggleColorScheme()}
                leftIcon={colorScheme === 'dark' ? 'ðŸŒš' : 'ðŸŒž'}
            >
                {'Do you want to toggle theme?'}
            </Button>
        )}
    </DocsContext.Consumer>
)
```
